# Aadhaar Ops Dashboard

A React-based Command Center Dashboard for UIDAI Aadhaar operations analytics.

## Live Demo
https://uidai-69d55.web.app/

## Features
- **Macro View**: Geospatial Intelligence with state intensity visualization
- **Forecaster**: Predictive Resource Allocation with time-series analysis
- **Watchtower**: Anomaly Detection with traffic light system
